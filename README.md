# Deepfake Face Detection Web App

This project is a **Deepfake Detection Web Application** that determines whether an uploaded video is **REAL or FAKE** using a deep learning model.

The system extracts frames from the uploaded video, detects faces, and uses a **ResNeXt + LSTM neural network** to classify the video.

---

## Features

- Upload a video for deepfake detection
- Frame extraction from video
- Face detection in frames
- Deep learning based classification
- Displays detected faces and processed frames
- Simple web interface built with Flask
